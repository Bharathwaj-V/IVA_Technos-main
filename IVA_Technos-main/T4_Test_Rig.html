T4 – TEST RIG

Test Jig designed to test the controller board used in Robot to clean solar panel. Jig have to test the controller as per the test procedure given by customer. The complete product is done by our group of teams which includes Mechanical Engineer, Electrical Engineer and Software developer. Test Jig tests data points for sensors, encoders, RF frequency measurement, Oscillometer to check the angle measurement.
